# SENTRON Home Assistant Integration

Integration für SENTRON Modbus Geräte (z. B. Siemens PAC Serie).

## Features
- Modbus TCP Verbindung
- Register auslesen
- UI Konfiguration (Config Flow)

## Installation (HACS)
1. HACS öffnen
2. Benutzerdefiniertes Repository hinzufügen
3. Dieses Repo auswählen
4. Integration installieren
5. Home Assistant neu starten

## Konfiguration
- IP Adresse
- Port (Standard 502)
- Register

## Unterstützte Geräte
- Siemens SENTRON PAC Serie (PAC2200, PAC3200, etc.)
